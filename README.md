# Web Development Project 1 - *Music Events Board*

Submitted by: **Rachel Villacorta**

This web app: A community site that showcases upcoming live music events and concerts in New York City, with a focus on discovering alternative genres and smaller artists alongside big names. Users can explore events via a responsive layout, see dates, locations, descriptions, and click links for more information.

Time spent: 8 hours spent in total
